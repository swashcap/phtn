Section 6 Variables

phtn relies on CSS variables for configuration of a a few properties. The variables are set on :root, and can easily be adjusted on a global or local level.

Colors

  • --a, --a2, --a3: Accent colors
  • --b: Black color
  • --g, --g2, --g3, --g4: Gray colors
  • --w: White color

Measurements

  • --br: Border radius
  • --s: Spacing unit
